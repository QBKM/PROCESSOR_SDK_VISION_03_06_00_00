GMACSW_NSP API Reference

Introduction

This documentation provides information on how to use and interact with the GMACSW_NSP driver for the 3-port Gigabit Switch peripheral (the GMACSW) available on a number of Texas Instruments embedded processors. The public interface exposed by the driver are used by the various networking stacks that send or receive data via the Ethernet hardware.

The driver, as currently written, sets the GMACSW hardware to act as a a simple MAC. It does not expose or make use of any significant switch functionality.
